# Implementation Plan: Course Content View

**Branch**: `007-content-view` | **Date**: 2026-05-07 | **Spec**: [spec.md](./spec.md)

---

## Summary

Build the Course Content View page (`dashboard/courses/:courseId/content`) — a standalone Angular component that fetches and displays all content items for a specific course via `GET /api/Content/course/{courseId}`, renders them as expandable cards with inline attachment lists, and provides permission-gated Edit (inline) and Delete (content + attachment) actions. The `ContentService` is created with all 7 API methods (5 for this cycle, 2 as stubs for the next).

---

## Technical Context

**Language/Version**: TypeScript 5.x / Angular 17 (Standalone Components)
**Primary Dependencies**: Angular HttpClient, RxJS, SweetAlert2, Bootstrap 5, Bootstrap Icons
**Storage**: N/A (stateless; data fetched from REST API per page visit)
**Testing**: Manual smoke test (T024-equivalent)
**Target Platform**: Web browser (desktop-first, responsive via Bootstrap grid)
**Performance Goals**: Content list renders within 2 seconds under normal network conditions
**Constraints**: No NgModule; Bootstrap 5 is the sole CSS framework; Scope-Lock enforced
**Scale/Scope**: Single-page component; expected ≤50 content items per course

### Service: `ContentService`

Full skeleton in `quickstart.md`. Methods:

| Method | HTTP | Endpoint | Cycle |
|--------|------|----------|-------|

| `getContentById(contentId)` | GET | `/api/Content/{contentId}` | ✅ This |
| `updateContent(id, title, body)` | PUT | `/api/Content/{contentId}` | ✅ This |
| `deleteContent(contentId)` | DELETE | `/api/Content/{contentId}` | ✅ This |

### Component: `ContentViewComponent`

**State**:
- `expandedIds: Set<number>` — collapsed by default (empty)
- `editingIds: Set<number>` — no card in edit mode on load
- `editFormData: Map<number, {title, body}>` — populated on `startEdit()`

**Methods**:
- `ngOnInit()` — read `courseId`, set permission flags, call `loadContent()`
- `loadContent()` — `GET /api/Content/course/{courseId}` → populate `contentList`
- `goBack()` — `Router.navigate(['/dashboard/courses'])`
- `toggleExpand(id)` — `expandedIds.has(id) ? expandedIds.delete(id) : expandedIds.add(id)`
- `startEdit(item)` — add to `editingIds`; populate `editFormData`
- `cancelEdit(id)` — delete from both sets silently
- `saveEdit(id)` — `PUT` → update card in `contentList` in-place → close form
- `deleteContent(id)` — Swal confirm → `DELETE` → `contentList.splice(i, 1)`
- `deleteAttachment(contentId, attachmentId)` — Swal confirm → `DELETE` → splice from `item.contentAttachments`
- `openAttachment(fileUrl)` — `window.open(fileUrl, '_blank')`
- `getAttachmentIcon(contentType)` — returns Bootstrap Icon class string

**Template key states**:
- Loading: spinner centered
- Error: red alert banner (distinct from empty state)
- Empty: illustrated empty-state card with message
- Populated: accordion card list
