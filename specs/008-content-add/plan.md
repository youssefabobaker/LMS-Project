## Summary

Implement a Bootstrap modal (id=`contentAddModal`) hosted in `content-view.component.html` that allows users with the `Content:add` permission to create a new course content item and sequentially upload PDF/MP4 attachments. The modal follows the same hosting pattern as `courseModal` / `assessmentModal` / `enrollmentModal` in `course-view.component.html`. The new `ContentAddComponent` (standalone) is declared inside the modal, emits a `(contentCreated)` event on success, and `ContentViewComponent` prepends the returned `Content` object to `contentList` — no page reload required.

The `ContentService.createContent()` and `ContentService.addAttachments()` stubs are already fully implemented; no service changes are needed.

## Component Design: ContentAddComponent

### Inputs

| Input | Type | Purpose |
|---|---|---|
| `courseId` | `number` | Passed from parent; used in Step 1 API call |

### Outputs

| Output | Type | Purpose |
|---|---|---|
| `contentCreated` | `EventEmitter<Content>` | Emits the newly created `Content` on full success |
| `modalDismissed` | `EventEmitter<void>` | Emits when Cancel is clicked (parent may use to reset state) |

### Component State

```typescript
title = '';
body  = '';
stagedFiles: StagedFile[] = [];

isSubmitting   = false;
submitError    = '';
retryMode      = false;          // true after Step 2 failure — hides normal submit, shows Retry
createdContentId: number | null = null;  // persisted after Step 1 success
```

### Key Methods

| Method | Responsibility |
|---|---|
| `onFilesSelected(event)` | Reads `FileList`, filters by MIME, filters by size ≤ 500MB, appends valid `StagedFile` objects to `stagedFiles[]` |
| `removeFile(index)` | Splices file out of `stagedFiles[]` |
| `formatSize(bytes)` | Returns human-readable string (B / KB / MB) |
| `getFileIcon(mimeType)` | Returns Bootstrap Icon class: `bi-file-earmark-pdf-fill` or `bi-play-circle-fill` |
| `submit()` | Validates form → calls Step 1 → on success calls Step 2 if files → emits `contentCreated` |
| `retryUpload()` | Calls Step 2 only with `createdContentId` — skips Step 1 |
| `cancel()` | Resets all state; emits `modalDismissed` |
| `resetForm()` | Called internally after success to prepare modal for next use |

## Sequential API Flow

```
User clicks "Save Content"
  │
  ├─ [Validate] title empty OR body empty?
  │     └─ YES → show validation error, stop
  │
  ├─ isSubmitting = true; submitError = ''
  │
  ▼
STEP 1: ContentService.createContent(courseId, title, body)
  │
  ├─ ERROR → submitError = message; isSubmitting = false; STOP
  │
  └─ SUCCESS → createdContentId = response.id
        │
        ├─ stagedFiles.length === 0?
        │     └─ YES → emit contentCreated(step1Response); close modal; toast
        │
        ▼
  STEP 2: ContentService.addAttachments(createdContentId, files[])
        │
        ├─ ERROR → retryMode = true; submitError = message; isSubmitting = false
        │           (modal stays open; title/body pre-populated; Retry Upload button shown)
        │
        └─ SUCCESS → emit contentCreated(step2Response); close modal; toast
```

## Parent Wiring: ContentViewComponent

### Template additions (`content-view.component.html`)

```html
<!-- Modal Host — append near end of template, before closing </div> -->
<div class="modal fade" id="contentAddModal" tabindex="-1"
     aria-labelledby="contentAddModalLabel" aria-hidden="true">
  <div class="modal-dialog modal-lg modal-dialog-centered modal-dialog-scrollable">
    <div class="modal-content rounded-3 overflow-hidden shadow-lg">
      <app-content-add
        [courseId]="courseId"
        (contentCreated)="onContentCreated($event)"
        (modalDismissed)="closeContentModal()">
      </app-content-add>
    </div>
  </div>
</div>
```

### TypeScript additions (`content-view.component.ts`)

```typescript
// Imports to add:
import { ContentAddComponent } from '../content-add/content-add.component';
// Add ContentAddComponent to @Component imports array

// New properties:
private contentAddModalInstance: any = null;

// Wire the stub:
onAddContent(): void {
  const el = document.getElementById('contentAddModal');
  if (el) {
    this.contentAddModalInstance = (window as any).bootstrap.Modal.getOrCreateInstance(el);
    this.contentAddModalInstance.show();
  }
}

// New handler:
onContentCreated(newItem: Content): void {
  this.contentList = [newItem, ...this.contentList];
  this.closeContentModal();
}

closeContentModal(): void {
  this.contentAddModalInstance?.hide();
}
```
